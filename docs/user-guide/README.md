# Ewoc User Guide

These pages are the public user-facing guide for the Android app and desktop
workout editor. They live in the repository so app behavior and guidance can be
updated together.

## Pages

- [Getting Started](getting-started.md)
  - First-session path, trainer setup, workout-mode choice, and basic next steps.
- [Desktop Editor](desktop-editor.md)
  - The first draft for building, previewing, validating, and exporting structured workouts.
- [Troubleshooting](troubleshooting.md)
  - The first support-facing guide for common setup, connectivity, guided-ride,
    and file-import problems.

## Planned Next Pages

- `Importing Workouts`
  - File-format coverage, supported paths, and structured import limitations.

## Writing Rules

- Keep user-facing guidance short, direct, and action-oriented.
- Prefer "what to do next" over deep technical explanation.
- Keep the app as the place for contextual microcopy and the guide as the place for fuller walkthroughs.
- Avoid promising features or workflows that are not already true in the app.
