## Planned Next Pages

## Writing Rules For This Draft Set

- Keep user-facing guidance short, direct, and action-oriented.
- Prefer "what to do next" over deep technical explanation.
- Keep the app as the place for contextual microcopy and the website as the place for fuller walkthroughs.
- Avoid promising features or workflows that are not already true in the app.
